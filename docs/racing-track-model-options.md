# Racing-track model options

## Selected and imported

The game now uses
[Race track/Karting Track based on South Garda](https://sketchfab.com/3d-models/race-trackkarting-track-based-on-south-garda-32c21042ba144ce9bd2822a88d5b54ec)
by Mauro3D. The supplied glTF contains 24,483 triangles and is licensed under
CC BY 4.0. Its original license is stored with the asset.

All three options are downloadable from Sketchfab under the Creative Commons
Attribution license. Keep the author credit and model link in the game credits
if a model is added.

## Recommended: Track by Mauro3D

- Model: https://sketchfab.com/3d-models/track-5f5e9454fd59436e8d0dd38df9ec83c4
- Author: Mauro3D
- Size: 1,400 triangles / 1,200 vertices
- License: CC BY
- Best use: the main online circuit. It is explicitly tagged game-ready and is
  light enough for the web build.

## Gravel Pit race track by Tomaga

- Model: https://sketchfab.com/3d-models/gravel-pit-race-track-922bb103d83a43c3b5c89f6446753f67
- Author: Tomaga
- Size: 49,500 triangles / 43,500 vertices
- License: CC BY
- Best use: a rally or off-road test facility. The author specifically permits
  racing-game use.

## Lowpoly Racetrack by FSkala

- Model: https://sketchfab.com/3d-models/lowpoly-racetrack-a2bd81e85612454c8f978f38f1aae0f2
- Author: FSkala
- Size: 106,200 triangles / 55,700 vertices
- License: CC BY
- Best use: a detailed motorsport venue or a source kit for track props. It
  should be optimized before use in the browser build.
